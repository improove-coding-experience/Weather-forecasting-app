# Weather-forecasting-app
A weather forecast app that shows current weather, temperature, and future updates
 Using Git Command Line

 # 1. Initialize git repository (if not already done)
git init

# 2. Add all files (except those in .gitignore)
git add .

# 3. Commit your changes
git commit -m "Initial commit: Weather Forecast App"

# 4. Create a new repository on GitHub (via website)
# Then link it to your local repository:
git remote add origin https://github.com/YOUR_USERNAME/Weather_forcarst_APP.git

# 5. Push to GitHub
git push -u origin main
